# Samsung Frame Art Rotator

A Home Assistant add-on that rotates images from an Immich shared album on a
Samsung Frame TV once a day. Runs entirely on your local network.

For full documentation, see **[DOCS.md](DOCS.md)** (also shown in the
Home Assistant add-on detail page).

## Quick start

1. Add this repository to your Home Assistant add-on store:
   `https://github.com/chermeschaterne/ha-samsung-frame-art-rotator`
2. Install the add-on
3. Configure (Immich share URL, TV IP + MAC, rotation time, etc.)
4. Start
